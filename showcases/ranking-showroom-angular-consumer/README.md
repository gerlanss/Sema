# Ranking Showroom Angular Consumer

Este showcase companion abre a trilha oficial `angular-consumer` da Sema usando o mesmo showroom de ranking ja provado no backend Flask.

## Slice oficial

- `consumer bridge` canonico em `src/app/sema_consumer_bridge.ts`
- configuracao de rotas lazy em `src/app/app.routes.ts`
- feature folders em `src/app/features/**`
- `impl` apontando so para bridge exportado

## Limite honesto

Isto nao promete geracao completa de UI nem visual drift. O slice aqui e `consumer bridge + route config surfaces`.

# Ranking Showroom Flutter Consumer

Este showcase companion abre a trilha oficial `flutter-consumer` da Sema usando o mesmo showroom de ranking ja provado no backend Flask.

## Slice oficial

- `consumer bridge` canonico em `lib/sema_consumer_bridge.dart`
- rotas consumer em `lib/router.dart`
- superfícies rastreaveis em `lib/screens/**`
- `impl` apontando so para bridge exportado

## Limite honesto

Isto nao promete geracao completa de UI nem visual drift. O slice aqui e `consumer bridge + router/screens`.

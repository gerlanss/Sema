# Ranking Showroom Stack Next.js

Este showcase junta o backend Flask oficial com o consumer oficial `Next.js consumer` na mesma raiz de operacao da CLI, sem duplicar codigo e sem gambiarra de contexto.

## O que ele conecta

- `../ranking-showroom`
- `../ranking-showroom-consumer`

## Fluxo rapido

```bash
sema inspecionar . --json
sema drift --json
```

O objetivo aqui e provar que a mesma raiz de trabalho consegue operar contrato backend e contrato consumer ao mesmo tempo.
